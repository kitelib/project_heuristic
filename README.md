Этот проект реализует алгоритмы планирования пути RRT (Rapidly-exploring Random Tree) и RRT-Connect для роботизированного манипулятора в двумерной среде с препятствиями. Код включает моделирование манипулятора с несколькими звеньями, создание среды с препятствиями различных форм, планирование траектории и визуализацию процесса.

## Установка и запуск

1. **Клонирование репозитория:**
   ```bash
   git clone https://github.com/kitelib/project_heuristic.git
   cd project_heuristic

2. **Настройка виртуального окружения (рекомендуется):**

```bash
  python -m venv .venv
  source .venv/bin/activate  # для Linux/Mac
  .venv\Scripts\activate     # для Windows

3. **Запуск одного из конфигурационных скриптов:**

```bash
python cfgs/run_1.py
### Входные и выходные данные

**Входные данные** 
Конфигурационные файлы в папке cfgs/ (например, run_1.py) содержат параметры алгоритмов и условия экспериментов.

Возможность настройки параметров через аргументы командной строки или прямую модификацию скриптов.

Выходные данные
Визуализации в формате GIF сохраняются в папке outs/ (например, test_1_try271844.gif).
Использование
Отредактируйте нужный конфигурационный файл в cfgs/, чтобы изменить параметры алгоритма.

После конфигурации cfgs/run_X.py запустить ячейки ноутбука proj.ipynb, отвечающие за работу алгоритмов и визуализацию результатов. При необходимости запустить ячейку бенчмаркинга
